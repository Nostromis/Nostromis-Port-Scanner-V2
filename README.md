# Nostromis-Port-Scanner-V2
This is a Port scanner that was made for a cyber security project it contains everything you need from multi threading to warning on input errors.

You will be prompted with four inputs to fulfill
# First IP:
You will be asked to enter the specified IP (Make sure its IPv4)
# Second Minimum_Port:
You will be asked to enter the starting port
# Third Maximum_Port:
You will be asked to enter the ending port
# Fourth Number_Of_Threads:
You will be asked to enter The amount of scans you want to be made at the same time use it according to your preference and Computer Hardware

After Fulfilling all of the inputs your scan will start on the specified IP and the ports will be between the range you enter (Minimum_Port To Maximum_Port)

And for ease of usage at the end you will be informed with the open ports so you don't have to look through all the logs
